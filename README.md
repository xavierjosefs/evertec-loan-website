# evertec-loan-website

## Entorno local

- Puerto de desarrollo: `9000`.
- API local pública: `http://localhost:4000/api`.

El MVP actual usa persistencia local para el website. `VITE_API_BASE_URL` queda preparado para la integración pública con la API. No coloques credenciales backend, MongoDB, OpenAI, S3/Spaces, correo, Stripe secret keys ni tokens en este proyecto.

## Variables públicas usadas

- `VITE_API_BASE_URL`
